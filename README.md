# 🚀 Novexa

> A production-inspired multi-vendor e-commerce platform built with the MERN stack.

Novexa is a full-stack e-commerce platform designed to simulate a real-world online marketplace where **Customers**, **Sellers**, **Drivers**, and **Administrators** interact through dedicated dashboards and role-based workflows.

The project focuses on scalability, modular architecture, secure authentication, efficient order processing, and responsive user experience. It demonstrates modern full-stack development practices by integrating a React frontend with a Node.js/Express backend and MongoDB database.
